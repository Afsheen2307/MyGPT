# MyGPT
MyGPT is a simple AI chatbot project (originally called ChatGPT).  
It uses an AI model (like OpenAI’s GPT) to answer questions and have conversations.

---

## Features
- Chat with an AI assistant  
- Easy to set up and run locally  
- Customizable prompts and settings  
- Simple and beginner-friendly code  

---

## Installation

1. Clone the repository  
   ```bash
   git clone https://github.com/your-username/MyGPT.git
   cd MyGPT
2. Install Dependencies
   npm install        # if using Node.js
   # or
   pip install -r requirements.txt   # if using Python
3. Add your API key
   Create a file named .env
   Add this line inside:
   OPENAI_API_KEY=your_api_key_here
4. Run the app
   npm start
   # or
   python main.py

## Usage
Once running, type your question or message to start chatting with MyGPT.
You can edit prompts or settings inside the config/ or prompts/ folders.

## Customize
Change the model or behavior in the code/config
Edit prompt templates for different types of conversations
Add your own features or UI
